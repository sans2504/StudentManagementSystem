## Usage

The application provides a menu-driven interface:
1. Add Student - Enter ID, Name, Grade
2. View All Students - Display all records
3. Update Student - Modify existing student
4. Delete Student - Remove student by ID
5. Search Student - Find student by ID
6. Exit - Close application
